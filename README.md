# 你的灵魂归处 — 米家专属人格测试 (Hoyoverse Personality Test)

> 20 道灵魂拷问，发现属于你的米游世界 ✧

这是一个单页面的人格测试应用，根据近乎灵魂拷问的选择题推断出最契合你的米哈游（HoYoverse）游戏世界（如《原神》、《崩坏：星穹铁道》、《绝区零》、《崩坏3》等）。同时，网页完全针对米家游戏经典的二次元科幻（Cyber-Tech / Glassmorphism）UI 风格进行了定制化视觉开发。

## ✨ 特性与亮点

- **米家原汁原味 UI 风格**：全页大量使用 CSS `clip-path`、磨砂玻璃 `backdrop-filter` 质感、暗黑底色与鎏金强调色配搭。
- **纯粹的 CSS 动画彩蛋**：包含隐藏的骨灰级“考古学家”结局——《FlyMe2theMoon》，并复刻了早期街机时代的 CRT 闪烁老电视滤镜，以及纯粹用 CSS `box-shadow` 和动画绘制的老式像素风满月夜。
- **极简的前端架构**：使用原生 Vanilla JS + HTML + CSS，以及 Vite 构建。无过重的框架开销，修改即所得。

## 🚀 运行方式

要对这个项目进行修改和本地预览：

```bash
# 1. 克隆代码
git clone https://github.com/AI1379/mihoyoti.git

# 2. 安装依赖
npm install 
# 或者使用 pnpm install / yarn install

# 3. 本地启动开发服务器
npm run dev
```

打开 `http://localhost:5175` 即可预览！
